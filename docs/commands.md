# Commands Reference

## Agent Manager Commands

| Command | Description |
|---|---|
| New Task | Start a new agent task |
| Pause | Pause the current agent |
| Resume | Resume a paused agent |
| Cancel | Cancel the current task |

## Editor Commands

Standard VS Code keybindings apply, plus Antigravity-specific commands for agent interaction.

## Slash Commands

> [!NOTE]
> Slash commands are available in the task input field to trigger built-in workflows.
